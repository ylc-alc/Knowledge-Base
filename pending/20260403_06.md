# Knowledge Summary
This note details how "hidden" web analytics data within HubSpot can be leveraged to automate lead property updates. By detecting keywords in page URLs, workflows can automatically populate missing information like 'Course of interest' [cite: 1].

# Extracted Details
- **Automated Property Setting**: Workflows can trigger based on specific values found in "hidden" data to fill missing CRM properties [cite: 1].
- **URL Keyword Detection**: If a lead's 'Course of interest' is missing, the workflow can scan the 'page seen URL' for specific keywords to assign the property [cite: 1].
- **Manual Agent Context**: Making web analytics properties visible in the lead record layout allows agents to manually extract keywords for additional context [cite: 1].
- **Intent Signals**: Web analytics properties are defined as "small intent signals with big potential" for lead qualification [cite: 1].

# Actionable Tips
- Configure HubSpot workflows to monitor "Page Seen" history for high-intent keywords [cite: 1].
- Add "Web analytics history" to the CRM record sidebar to provide sales agents with immediate context on lead interests [cite: 1].
