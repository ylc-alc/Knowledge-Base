# Knowledge Summary
This screenshot demonstrates a high-quality prompt structure for using Google Gemini to perform business analysis and data visualization within Google Sheets. It breaks down the prompt into four key components: state the 'what', state the 'goal', state the 'task', and specify the 'visualization' requirement [cite: Screenshot_20250904_222531_YouTube.jpg].

# Extracted Details
- **Prompt Components** [cite: Screenshot_20250904_222531_YouTube.jpg]:
    - **State the What**: Provide context on the project (e.g., "I am working on a business analysis for Netflix") [cite: Screenshot_20250904_222531_YouTube.jpg].
    - **State the Goal**: Define the specific objective (e.g., "analyze the company's financial performance in 2024") [cite: Screenshot_20250904_222531_YouTube.jpg].
    - **State the Task**: Instructions for the AI (e.g., "based on the data attached, create a learning agenda with 5-8 questions") [cite: Screenshot_20250904_222531_YouTube.jpg].
- **Learning Agenda Requirements** [cite: Screenshot_20250904_222531_YouTube.jpg]:
    - Explain why each question is important (ties back to the goal) [cite: Screenshot_20250904_222531_YouTube.jpg].
    - Identify potential business opportunities [cite: Screenshot_20250904_222531_YouTube.jpg].
    - Define how the answer will be measured [cite: Screenshot_20250904_222531_YouTube.jpg].
- **Visualization Instruction**: Ask for a specific Gemini prompt to visualize the data in Google Sheets, keeping it simple and starting with "Visualize..." without specifying chart types to allow the AI to choose the best option [cite: Screenshot_20250904_222531_YouTube.jpg].

# Actionable Tips
- Use the "What -> Goal -> Task" framework for all data-heavy prompts to ensure clarity [cite: Screenshot_20250904_222531_YouTube.jpg].
- When asking for visualizations, use the "Visualize..." keyword to trigger built-in Google Sheets capabilities [cite: Screenshot_20250904_222531_YouTube.jpg].
