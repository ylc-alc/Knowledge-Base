# Knowledge Summary
A guide for creating high-engagement social media content, specifically focused on tool reviews and problem-solving hook structures.

# Extracted Details
- Tool Introduction Template: Includes Name, Function, URL, Price, and Pros/Cons.
- Content Framework: Hook -> Problem -> Cause -> Solution -> Action.
- Hook Focus: Stresses that the first line determines if users stop scrolling.
- Visuals: Recommends using clear layouts or lists for readability.

# Actionable Tips
- Use the 'Hook-Problem-Solution' framework for educational posts.
- Standardize tool reviews with a fixed template to build trust and clarity.
