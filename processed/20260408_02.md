# Knowledge Summary
Introduction to two specific "Skills" or tools designed to streamline the web design process and refine AI-generated user interfaces to look more professional and less generic.

# Extracted Details
- **UI-UX-Pro-Max**:
    - Includes 161 color schemes.
    - Features 57 font pairings.
    - Contains a library of 99 UX best practices.
    - Purpose: Establishes a complete design system before writing any code.
- **Impeccable**:
    - Specifically targets "AI design taste" (fixing generic-looking AI outputs).
    - Provides 20 precision commands.
    - Key commands mentioned: `/bolder` (bold/brave design), `/quieter` (steady/calm design), and `/delight` (adding memorable details).
    - Purpose: Refining specific sections of a UI that has already been drafted.

# Key Takeaways
- Tools are shifting from "generating" to "refining" and "systematizing" to avoid the common pitfalls of raw AI-generated designs.
- Consistency is prioritized through pre-built color and font libraries (UI-UX-Pro-Max).

# Actionable Tips
- Use UI-UX-Pro-Max at the start of a project to prevent "blank canvas" syndrome and ensure consistency.
- Apply Impeccable commands like `/quieter` if a design feels too busy or "over-designed" by an AI model.
