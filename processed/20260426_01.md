# Knowledge Summary
A fundamental framework for structured communication and storytelling used to define a problem and its resolution.

# Extracted Details
- **Situation**: Defines the current state or context ('Here's where we are').
- **Complication**: Highlights the obstacle or conflict that disrupts the situation ('Here's the obstacle/conflict').
- **Question**: Identifies the core problem that needs solving to move forward ('What do we need to answer to move forward?').
- **Answer**: Provides the solution or resolution to the question ('Here's the resolution').

# Key Takeaways
- Use SCQA to ensure your presentations or writing have a clear logical flow.
- It helps in aligning the audience on the 'why' before giving the 'how'.
