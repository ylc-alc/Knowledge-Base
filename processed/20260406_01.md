# Knowledge Summary
Cloudflare has introduced a Crawler API, a significant pivot for a company traditionally known for anti-bot and anti-crawling technology. This tool allows users to crawl entire websites and retrieve content in structured formats like Markdown or JSON.

# Extracted Details
- Cloudflare released a Crawler API that can crawl entire websites in a single call. [cite: 1]
- The API returns content in HTML, Markdown, or JSON formats. [cite: 1]
- It supports JavaScript rendering, meaning it can crawl dynamic sites built with React or Vue. [cite: 1]
- It features incremental crawling, which identifies and crawls only updated pages to save time and costs. [cite: 1]
- Data can be returned directly as structured JSON. [cite: 1]
