# Knowledge Summary
A deep dive into the architecture of LLM agents, focusing on the distinction between the 'intelligence' (the model) and the 'harness' (the code that creates a stable environment for the model to work).

- Key components of an agent harness include the agent loop, tools, planning, subagents, context compression, and multi-agent isolation. [cite: 5]
